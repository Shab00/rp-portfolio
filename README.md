# Personal Portfolio Project

This repository contains the source code for a personal portfolio website built with Django, a high-level Python web framework. The project is designed to showcase personal projects, achievements, and skills.

## Features

- **Dynamic Project Display:** Projects are dynamically displayed from the database, allowing easy updates and management.
- **Contact Form:** Includes a contact form that sends emails, making it easy for visitors to get in touch.
- **Responsive Design:** The website is designed to be responsive, ensuring it looks great on all devices.

## Getting Started

### Prerequisites

Before you begin, ensure you have met the following requirements:
- You have installed Python 3.8 or higher.
- You have a Django-compatible database installed (SQLite is provided by default).
